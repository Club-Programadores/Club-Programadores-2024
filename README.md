# Programador@s HUB

Base de datos de programadores que participan y proyecto del Club de Programadores del Centro Universitario de Vicente Lopez.

## Comandos:

- Install Dependencies
```node
npm i
````
- Run
```node
npm run dev
````
- Build
```node
npm run build
````
